Privacy

This website does not use cookies, does not collect personal data, and does not use tracking scripts that follow you across the web.

For anonymous usage statistics, this site uses GoatCounter, a privacy-friendly, open-source analytics tool. GoatCounter does not use cookies, does not collect personal data, and does not track visitors across sites. It records only basic, non-identifiable information such as the page visited, the referrer, and the browser type. IP addresses are not stored.

No data is shared with third parties.
